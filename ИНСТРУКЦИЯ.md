# Инструкция по боту — запись кандидатов на созвон

Кандидат приходит по ссылке, сам выбирает удобное время, получает напоминания и контакт куратора. Всем управляете через меню `/admin`.

---

## 1. Что делает бот (путь кандидата)

1. **Переход по ссылке** — кандидат открывает `/start` (можно с меткой источника). Бот показывает вакансию и ближайшие созвоны с кнопками.
2. **Выбор времени** — жмёт удобный слот, сразу получает ссылку на созвон и имя закреплённого куратора. Если ничего не подходит — жмёт «Ни одно время не подходит» и получает контакт HR.
3. **Подтверждение** — «Да, буду» → записан. «Не смогу» → бот предложит другие слоты, а на следующий созвон пришлёт повторное приглашение.
4. **Напоминания** — подтвердившим приходят пуши: за 1 час, за 15 минут (со ссылкой) и в момент старта. Созвон открывается в браузере, ставить ничего не нужно.
5. **Контакт куратора** — через несколько минут после начала созвона каждому участнику автоматически приходит контакт его куратора.

---

## 2. Меню администратора (`/admin`)

Отправьте боту `/admin`. Меню доступно только заданным ID.

| Кнопка | Что делает |
|--------|-----------|
| 📅 **Назначить созвон** | Ввод даты (`ДД.ММ.ГГГГ`), времени (`ЧЧ:ММ`, МСК) и ссылки. Бот сам разошлёт приглашения новым кандидатам и тем, кто отказался/промолчал ранее. |
| 📝 **Редактировать вакансию** | Единый текст вакансии, который видят все кандидаты на входе. |
| 👥 **Кураторы** | Список кураторов, между которыми кандидаты делятся поровну (50/50). Каждого можно включить/выключить. Кандидат закрепляется за одним куратором навсегда. |
| 🆘 **Контакт для подбора времени** | Контакт HR для тех, кто нажал «Ни одно время не подходит». |
| 📊 **Статистика** | Воронка: зашло / подтвердило / отказалось / получило контакт. Есть срез по созвону и по источнику трафика. |
| 🔗 **Реферальные ссылки** | Ссылки вида `t.me/бот?start=источник` для разных каналов (Instagram, HH, Авито и др.). Источник запоминается за кандидатом. Можно добавить свой. |
| 📥 **Экспорт в Excel** | Выгрузка всех кандидатов со статусами и источником в `.xlsx`. |
| 📢 **Рассылка** | Сообщение всем кандидатам или только участникам конкретного созвона. Перед отправкой — предпросмотр. |
| 🚫 **Чёрный список** | Загрузка `.xlsx` со списком username — этим людям бот перестанет отвечать. Можно выгрузить все username из бота. |
| ✏️ **Редактировать созвон** | Изменить время и/или ссылку созвона. Участникам придёт уведомление. |
| ❌ **Отменить созвон** | Отмена: напоминания снимаются, участникам приходит уведомление с предложением выбрать другое время. |

---

## 3. Как распределяются кураторы

Когда кандидат выбирает время, бот назначает ему куратора по очереди — строго поровну. Дальше кандидат **всегда попадает к тому же куратору**, даже на другом созвоне.

- **Оба активны** — кандидаты идут 50 / 50.
- **Один выключен** — все новые идут ко второму.
- **Оба выключены** — используется запасной контакт менеджера.

> ⚠️ Включение/выключение куратора **не сбивает** очередь. Только полностью новый список кураторов начинает счёт заново.

---

## 4. Приглашения и повторы

Каждый кандидат получает не более **3 приглашений суммарно** (1 первичное + 2 повторных). Повтор на следующий созвон приходит тем, кто отказался или не отреагировал — пока лимит не исчерпан.

---

## 5. С чего начать (первый запуск)

1. Откройте `/admin` — если меню открылось, ваш ID добавлен как администратор.
2. **📝 Редактировать вакансию** — впишите текст, который увидят кандидаты.
3. **👥 Кураторы** — добавьте минимум одного.
4. **🆘 Контакт для подбора времени** — укажите своего HR.
5. **📅 Назначить созвон** — создайте и разошлите ссылку кандидатам.

---

*Время везде — московское (МСК). Все настройки меняются прямо в боте через `/admin`, без программиста.*
